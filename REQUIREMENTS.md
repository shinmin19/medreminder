# 💊 吃药提醒APP - 需求文档 v1.0

> 基于Medisafe、iCare、MyTherapy、DoseMed等头部APP功能调研，结合主人实际需求整理

---

## 一、项目概述

| 项目 | 说明 |
|------|------|
| **APP名称** | 待定（建议：药管家 / 吃药啦 / MedMinder） |
| **平台** | iOS（自签名） |
| **开发框架** | Flutter（跨平台预留，一套代码可扩展Android） |
| **数据存储** | 本地SQLite（隐私优先，无需后端服务器） |
| **最低系统** | iOS 16+（支持HealthKit和本地通知） |

---

## 二、核心功能模块

### 📦 模块1：药品管理

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 添加药品 | 药品名称（自定义输入）| ⭐⭐⭐ |
| 药品图标/颜色 | 可选预设药片图标+颜色区分 | ⭐⭐ |
| 剂量设置 | 每次服用量（如：1片、2粒、5ml）| ⭐⭐⭐ |
| 服药频率 | 每天N次 / 隔N天 / 每周N次 / 自定义周期 | ⭐⭐⭐ |
| 服药时间 | 可设置每天多个时间点（如早中晚）| ⭐⭐⭐ |
| 服用方式 | 口服/外用/注射等（备注用）| ⭐ |
| 备注说明 | 药品注意事项、饭前饭后等 | ⭐⭐ |
| 药品库存 | 剩余数量，低于阈值提醒补药 | ⭐⭐ |
| 有效期管理 | 药品过期日期，过期提醒 | ⭐⭐ |

### ⏰ 模块2：智能提醒（核心）

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 定时提醒 | 按设定时间推送本地通知 | ⭐⭐⭐ |
| 提前提醒 | 用药前N分钟推送通知（可自定义：5/10/15/30/60分钟）| ⭐⭐⭐ |
| 间隔提醒 | 如果到了吃药时间没吃，每隔N分钟重复提醒 | ⭐⭐⭐ |
| 贪睡功能 | 点击"稍后提醒"，延迟N分钟再提醒 | ⭐⭐⭐ |
| 通知样式 | 弹窗 + 声音 + 震动（可单独开关）| ⭐⭐⭐ |
| Critical Alert | iOS关键提醒，即使静音也能响（iOS 14+）| ⭐⭐ |
| 勿扰适配 | 仅在用户设定的时段内提醒 | ⭐⭐ |

### ✅ 模块3：用药打卡

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 一键打卡 | 收到通知后直接在通知上标记"已服用" | ⭐⭐⭐ |
| 补打卡 | 在APP内手动补记过去的用药 | ⭐⭐⭐ |
| 漏服标记 | 自动标记未打卡的为"漏服" | ⭐⭐⭐ |
| 批量打卡 | 一次服用多种药时可批量全部标记 | ⭐⭐⭐ |
| 打卡记录 | 查看每天/每周/每月的服药记录 | ⭐⭐⭐ |

### 📅 模块4：日历视图

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 月视图日历 | 显示整月用药情况 | ⭐⭐⭐ |
| 颜色标记 | 🟢全部服用 🟡部分漏服 🔴全部漏服 ⚪无药日 | ⭐⭐⭐ |
| 日详情 | 点击日期查看当天具体服药明细 | ⭐⭐⭐ |
| 周视图 | 快速浏览本周用药情况 | ⭐⭐ |
| 用药连续天数 | 显示当前连续服药天数（打卡激励）| ⭐⭐ |

### 📊 模块5：统计报告

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 依从率 | 总体服药依从率（已服/应服 × 100%）| ⭐⭐⭐ |
| 趋势图 | 按周/月展示服药趋势折线图 | ⭐⭐ |
| 单药统计 | 每种药的单独服药率 | ⭐⭐ |
| 导出报告 | 导出PDF/CSV，可发给医生 | ⭐⭐ |

### 🔔 模块6：补药提醒

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 库存追踪 | 记录每种药的总数量 | ⭐⭐ |
| 自动计算 | 根据服药频率自动计算剩余可用天数 | ⭐⭐ |
| 补药提醒 | 库存低于X天时推送"该买药了" | ⭐⭐ |

## 四、交互设计要点

### 🎨 设计原则
- **简洁**：参考iCare的简约风，不要过度设计
- **大字体**：考虑到可能有老年用户，关键按钮要大
- **打卡操作≤2步**：通知点一下就能完成打卡
- **药品颜色区分**：不同药品用不同颜色药片图标

### 📱 页面结构
```
首页（今日用药）
├── 今日待服药物列表
├── 一键打卡按钮
├── 服药进度条
└── 连续打卡天数

药品管理
├── 药品列表
├── 添加/编辑药品
└── 药品详情

日历
├── 月视图（颜色标记）
├── 日详情
└── 周视图切换

统计
├── 依从率饼图
├── 趋势折线图
└── 导出报告

设置
├── 提醒设置（提前时间/间隔/声音/震动）
├── 通知偏好
├── 数据管理（备份/导出/删除）
└── 关于
```

## 五、技术方案

### 推荐技术栈
| 组件 | 方案 |
|------|------|
| **框架** | Flutter 3.x + Dart |
| **本地通知** | flutter_local_notifications |
| **本地数据库** | sqflite（SQLite）|
| **日历组件** | table_calendar |
| **图表** | fl_chart |
| **状态管理** | Provider / Riverpod |
| **HealthKit** | health package |
| **打包签名** | Xcode自签名（Free Team） |

### 自签名流程
1. Apple ID注册（免费账号即可）
2. Xcode创建Flutter项目
3. Signing & Capabilities → 选Free Personal Team
4. Build → 直接安装到iPhone（USB）
